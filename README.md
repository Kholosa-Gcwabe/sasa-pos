## Sasa POS System

A restaurant Point of Sale (POS) prototype designed to digitize order management for small businesses.

## Features
- Digital order processing (counter → kitchen)
- Real-time order tracking
- Automated receipt generation
- Responsive interface for cashiers

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

## How to Run
1. Clone the repository
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open index.html in browser
